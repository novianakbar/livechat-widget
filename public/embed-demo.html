Demo Embed Livechat Widget

Online Single Submission (OSS) - Perizinan Berusaha

1. Embed Otomatis (Direkomendasikan)

Widget akan otomatis muncul di pojok kanan bawah halaman:

<!-- Cukup tambahkan script ini ke halaman HTML Anda --> <script src="./livechat-oss-widget.umd.js"></script>
Catatan: Widget akan muncul secara otomatis tanpa perlu konfigurasi tambahan.

2. Embed dengan Konfigurasi

Untuk mengkustomisasi widget, tambahkan konfigurasi sebelum script:

<script> window.LivechatOSSConfig = { primaryColor: '#3B82F6', widgetTitle: 'Bantuan OSS Perizinan', welcomeMessage: 'Selamat datang! Bagaimana kami bisa membantu perizinan berusaha Anda?', position: 'bottom-right', // bottom-left, top-right, top-left autoOpen: false, allowFileUpload: true }; </script> <script src="./livechat-oss-widget.umd.js"></script>

3. Embed Manual (Kontrol Penuh)

Untuk kontrol lebih detail, gunakan JavaScript API:

<div id="my-livechat"></div> <script src="./livechat-oss-widget.umd.js"></script> <script> // Buat instance widget const widget = new LiveChatWidget({ primaryColor: '#3B82F6', widgetTitle: 'Bantuan OSS', position: 'bottom-right' }); // Mount ke element tertentu widget.mount('#my-livechat'); </script>
